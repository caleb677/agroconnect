🌽 AgroConnect — Setup Guide

Murang'a County Agricultural Platform · Follow these steps in order

✅ Frontend — Ready ✅ Backend files — Ready ⚠️ Database — Needs Supabase URL ⚠️ M-Pesa — Needs Safaricom keys ⚠️ AI — Needs Anthropic key
1

Create Free Database on Supabase

Takes 3 minutes · Completely free

1. Open https://supabase.com → Sign up (use Google)

2. Click "New Project" → Name it agroconnect

3. Set a database password (write it down!)

4. Region: West EU (Ireland) or Singapore — closest to Kenya

5. Wait ~2 minutes for it to start

6. Go to Project Settings → Database → Connection string → URI tab

7. Copy the URL that looks like this:

postgresql://postgres:YourPassword@db.abcxyz.supabase.co:5432/postgres
⚠️ Replace [YOUR-PASSWORD] in the URL with the actual password you set in step 3.
2

Update Your .env File

Open backend/.env in Notepad or VS Code

Open the file AgroConnect_FULL/backend/.env and paste your Supabase URL:

DATABASE_URL=postgresql://postgres:YourPassword@db.abcxyz.supabase.co:5432/postgres JWT_SECRET=agroconnect-muranga-2025-caleb-secret-key-32chars ADMIN_EMAIL=calebmwasi22@gmail.com
✅ Save the file. That's enough to start — you can add other API keys (M-Pesa, AI, email) later.
3

Create Database Tables + Admin Account

Run these commands in your terminal (in the backend folder)

cd AgroConnect_FULL\backend node config/migrate.js

You should see: ✅ All tables created successfully.

node config/seed-admin.js

You should see: ✅ Super Admin created! Email: calebmwasi22@gmail.com

node config/seed-demo.js

This adds demo Extension Officers, Farmers, Buyers and Agrovets for testing.

4

Start Both Servers

Open TWO terminal windows

Terminal 1 — Backend:

cd AgroConnect_FULL\backend npm run dev

Should show: ✅ AgroConnect backend running on port 4000


Terminal 2 — Frontend:

cd AgroConnect_FULL\frontend npm run dev

Open http://localhost:5173 in your browser.

✅ Login with:

6

Deploy Online (Make it public)

Both free hosting options

🖥️ Backend → Render.com (FREE) 1. Push backend/ to GitHub
2. render.com → New Web Service
3. Build: npm install
4. Start: npm start
5. Add all .env variables
6. URL: agroconnect-api.onrender.com
🌐 Frontend → Vercel (FREE) 1. Push frontend/ to GitHub
2. vercel.com → New Project
3. Set VITE_API_URL to Render URL
4. Deploy automatically
5. URL: agroconnect.vercel.app
6. Add custom domain (optional)
⚠️ For M-Pesa to work live, update: MPESA_CALLBACK_URL=https://your-render-url.onrender.com/api/mpesa/callback
